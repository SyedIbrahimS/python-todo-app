A Todo App built with Python and Tkinter. Features include adding, deleting, editing, searching, marking tasks as completed, task counters, and file-based data saving.
# Todo App

A Todo App built using Python and Tkinter.

## Features
- Add tasks
- Delete tasks
- Edit tasks
- Search tasks
- Mark tasks as completed
- Count total and completed tasks
- Save tasks to a text file
- Load tasks automatically on startup
- Scrollbar support

## Technologies Used
- Python
- Tkinter
- File Handling
